NLCI Tamil fonts
================

This file describes the source files included with the NLCI Tamil fonts.
This information should be distributed along with the NLCI Tamil fonts and any derivative works.

master.gdl            - Graphite rules
master_src.sfd        - OpenType rules
other .sfd files      - glyph outline data for each font
thiruvalluvar_classes.xml - class information for the attachment point database
thiruvalluvar_unicode.xml - mapping from legacy to Unicode as used by ttfbuilder
TEL ThiruValluvar 2 Unicode CONV chart.xls - source for thiruvalluvar_unicode.xml
